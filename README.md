# NightCity Bank & Arcade
A fully offline, cyberpunk-themed mobile banking simulation and arcade app built with React Native and Expo.

NightCity Bank provides a futuristic immersive experience where users can manage their finances, invest in a dynamic stock market, take out loans, and play arcade games—all without ever needing an internet connection.

# Features
Immersive Banking: Manage credits, perform wire transfers, and track transaction history.

Dynamic Economy: A randomized stock market that fluctuates every few seconds, allowing users to build a portfolio.

# Arcade Zone:

Plinko: Physics-based dropping game with velocity and gravity simulation.

Blackjack: Fully functional card game against a simulated dealer.

Cyber Slots: RNG-based reel game with jackpot mechanics.

Spin & Win: A high-stakes wheel of fortune.

Service Management: Pay for "Energy," "Water," and "Net" utilities using the integrated bill payment system.

Offline-First: No server dependency. All data is persisted locally using AsyncStorage.

# Tech Stack
Framework: React Native / Expo

Language: TypeScript

State Management: React Context API

Persistence: AsyncStorage

Navigation: React Navigation

Styling: StyleSheet (Custom Cyberpunk Theme)
